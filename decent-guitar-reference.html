🎸 Decent Guitar

Project Reference — last updated June 2026

Overview

Decent Guitar is a personal web app for supporting acoustic guitar learning. It provides a song library with chord-annotated scrolling lyrics (similar to Ultimate Guitar's chord view), a chord fingering library, strumming pattern reference, and a daily practice module (in progress).

song_parts_order

Intermediate table. Assigns a song-specific ordering to part instances (e.g. Verse appears at position 2 and 4 in the same song). Unique constraint on (song_id, part_order).

ColumnTypeNotes
PK iduuid
FK song_iduuid→ songs, cascade delete
FK part_idint→ parts
part_orderint1-based sequence within a song

song_part_strumming

Overrides the song-level default strumming pattern for a specific part instance.

ColumnTypeNotes
FK song_parts_order_iduuidComposite PK with strumming_pattern_id
FK strumming_pattern_idint→ strumming_patterns

song_lyrics

One row per line — chord line or lyric line, distinguished by line_type. Lines are strictly ordered by line_order within a part (not paired or grouped — just sequential). Fixed-width text in chord lines is used to visually align chords above lyrics on display. A part may have chord-only or lyric-only lines (e.g. instrumental bars).

ColumnTypeNotes
PK iduuid
FK song_parts_order_iduuid→ song_parts_order, cascade delete
line_orderint1-based sequence within a part. Unique with song_parts_order_id.
lyric_texttextRaw line text, fixed-width spacing for chord alignment
line_typetext'chords' or 'lyrics' (check constraint). Comment type planned — see backlog.

Practice Module — Chord Change Cycling

The Practice tab's first module supports building and drilling a chord sequence at a set tempo.

  • Sequence builder: tap any chord from your library to add it to the sequence (repeats allowed, unbounded length). Tap a chord in the sequence to remove it.
  • BPM (default 60) and Beats per Chord (default 4) — both user-adjustable.
  • Metronome: Web Audio API click — higher pitch on beat 1 of each chord, lower pitch on beats 2+. Audio is silent if the device is hardware-muted (e.g. iPad side switch) — this is a device-level setting, not app-controllable.
  • Display: current chord name (large) + full string-by-string fingering table, beat-progress dots (fill 1→BPC across the current chord's full duration), and a "Next chord" preview shown throughout the current chord's measure — switching only when the chord actually changes, not pre-emptively on the last beat.

This is a UI-only module — no database table writes; it reads from the existing chords table.

Ultimate Guitar Text Import

Admin → Songs includes a paste-in text importer. Paste UG-style song text (title/artist header, Capo line, then [Part Name] markers with chord and lyric lines below each) and click Parse Text. The parser:

  • Extracts title/artist from a "Title Chords by Artist" header line
  • Extracts capo fret from a "Capo: N" line
  • Splits the body on [Part Name] markers
  • Classifies each line as chord or lyric — chord lines are lines where every whitespace-separated token matches a chord-name pattern (e.g. Em, A, G, D, Cmaj7, F#m)
  • Matches part names to the parts reference table where possible (e.g. "Verse 1" → "Verse"); unmatched parts are left for manual selection
  • Populates the existing parts/lines builder UI — fully editable before saving

This is a UI pre-fill only; it does not touch the database directly. Save Song still goes through the normal save path.
